# Industry Resume Template

A compact one-page LaTeX resume template for industry applications. The included example uses Albert Einstein as a public sample profile, so the repository contains no private candidate information.

![Resume preview](assets/resume.png)

## Files

- `resume.tex` - editable LaTeX source
- `resume.pdf` - compiled one-page PDF
- `assets/resume.png` - preview image used in this README

## Build

```bash
pdflatex -interaction=nonstopmode resume.tex
```
